# WPF Translation App

A simple WPF desktop application for translating text between different languages using the LibreTranslate API.

---

## Features

- **Real-time text translation**
- **Support for multiple languages**
- **Modern user interface**
- **Easy-to-use design**

---

## Technologies Used

- **C# / WPF**
- **.NET Framework**
- **MVVM Architecture**
- **LibreTranslate API**
